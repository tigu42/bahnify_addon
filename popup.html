Bahnify Extension

Verspätungsstatistiken für Züge auf www.bahn.de

Datenstand ab 21.03.2024

Aktuell werden über 130 der größten Bahnhöfe Deutschlands gespeichert

Bahnify Extension Version 1.0.0

Die Reisedetails auf www.bahn.de werden durch diese Erweiterung verändert. Die hinzugefügten Statistiken werden aus den Fahrplan- und Fahrplanänderungsdaten der Timetables API des API Marketplace der Deutschen Bahn erzeugt. Die Daten der Timetables API unterliegen der Creative Commons Attribution 4.0 International (CC BY 4.0) Lizenz. Um eine effiziente Speicherung und einen schnellen Zugriff zu ermöglichen, wurden die Daten in ein stark reduziertes Format gebracht. Dieser veränderte Datensatz weist Schwächen auf: Alternativzüge werden in den Statistiken nicht berücksichtigt. Wenn die Weiterfahrt eines Zuges ausfällt, so kann auch dessen Ankunft als entfallen in die Statistik einfließen. Die Richtigkeit der anderen erzeugten Statistiken kann ebenfalls nicht garantiert werden. Bei Verbindungen, bei denen der Umstieg über Mitternacht geht, kann zurzeit keine Statistik erzeugt werden. Bahnify speichert keine Nutzungsdaten.
